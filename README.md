# Apuntes de Terraform

Este repositorio contiene una guía estructurada para el aprendizaje y dominio de **Terraform** como herramienta de Infraestructura como Código (IaC). El objetivo es centralizar conceptos, comandos y buenas prácticas organizados por niveles de complejidad.

## 📂 Contenido del Repositorio

* **[Nivel Básico](./README-basico.md):** Conceptos fundamentales, ciclo de vida inicial y comandos esenciales para desplegar recursos individuales.
* **[Nivel Intermedio](./README-intermedio.md):** Modularización, gestión de estados remotos, variables dinámicas y funciones.
* **[Nivel Avanzado](./README-avanzado.md):** Orquestación compleja, CI/CD, gestión de infraestructura multi-cloud, seguridad (Vault) y Testing.

## 🚀 Flujo de Trabajo General
1. `terraform init` -> 2. `terraform plan` -> 3. `terraform apply`
