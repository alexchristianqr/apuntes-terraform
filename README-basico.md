# Terraform - Nivel Básico

Enfoque en el flujo de trabajo estándar y la creación de recursos simples.

| Comando | Descripción |
| :--- | :--- |
| `terraform init` | Inicializa el directorio de trabajo, descarga proveedores y configura el backend. |
| `terraform validate` | Verifica que la sintaxis de los archivos `.tf` sea correcta. |
| `terraform plan` | Crea un plan de ejecución para previsualizar los cambios antes de aplicarlos. |
| `terraform apply` | Ejecuta las acciones propuestas en el plan para crear o modificar la infraestructura. |
| `terraform destroy` | Elimina todos los recursos gestionados por el proyecto actual. |
| `terraform fmt` | Formatea automáticamente el código siguiendo los estándares de HCL. |
| `terraform show` | Muestra el estado actual o un plan en un formato legible. |
